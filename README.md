# LPS Chromebook Tester v18

Static browser-based Chromebook testing dashboard.

Version 18 fixes:
- System & Browser Info no longer includes Memory Info, Local Storage Capacity, or Local Storage Remaining.
- Keyboard uses the simplified layout with the top function row displayed as blank visual keys and not tested.
- Keyboard test has Start and Stop buttons.
- Microphone test has a Stop Mic Test button.
- CSS and JS references include v18 cache-busting query strings for GitHub Pages / ChromeOS kiosk caching.

Upload/replace all files in the GitHub repository root: index.html, app.js, styles.css, README.md, manifest.json, and icon.svg.
