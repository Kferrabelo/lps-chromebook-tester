# LPS Chromebook Tester v17

Static browser-based Chromebook testing dashboard.

Version 17 updates:

- Removed Memory Info, Local Storage Capacity, and Local Storage Remaining from System & Browser Info.
- Added Start keyboard test and Stop keyboard test buttons.
- Added Stop mic test button.
- Updated the keyboard layout to match the newer reference image with the function/top row shown as blank keys, since those keys do not need to be tested.

Open `index.html` from a web server, HTTPS host, or localhost for best camera and microphone behavior.
